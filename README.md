Esta api tem o objetivo de criar uma simples lista de desejos, onde você pode estar colocando:

* Produto
* Descrição
* Link*
* Adquirido
* Imagem
  
Indico que você use um encurtador de link, pois links muito grandes não serão aceitos na api.
